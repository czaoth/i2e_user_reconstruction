i2e-user-reconstruction
========================

openFrameworks App demonstrating online human reconstruction using librecon.

Dependencies:
==================

openFrameworks addons:
* ofxXmlSettings
* ofxMSATimer
* ofxTimeMeasurements
* ofxUI

Other libaries:
* Boost 1.55
* Flann
* OpenNI2
* qHull
* Eigen3
* Qt 4.8.6
* VTK
* PCL
* UnitTest++
